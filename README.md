# Привет, я Руслан 👨‍💻

Добро пожаловать в мой GitHub! Здесь я делюсь проектами, которые создаю в процессе изучения фронтенд-разработки. Мой путь начался с обычного любопытства — "а как это работает?" — и постепенно превратился в серьёзное увлечение созданием интерфейсов.

## Мой подход к обучению

Мне кажется важным не просто изучать теорию, а сразу применять знания на практике. Поэтому каждый новый фреймворк или библиотеку я пробую в реальном проекте

**Мой текущий фокус:**

1. Углубление в React (оптимизация, кастомные хуки)
2. Изучение TypeScript для повышения надёжности кода
3. Практика алгоритмов на платформах типа Codewars
4. Улучшение навыков доступности (a11y) и семантической вёрстки
